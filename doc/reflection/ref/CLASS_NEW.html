Documentation

the CLASS_NEW macro is used to declare reflection support in a class. It must be placed on the public part of the class. This MACRO will allow creation of new object using newInstance() method on the Class
Parameters:
CLASS_NAME - name of this class.
SUPER_CLASS_NAME - name of the super class; if there is no base class, pass NullClass.
