#define PROPERTY

(TYPE, NAME)

The PROPERTY macro is used to declare an object's property.

Documentation

The PROPERTY macro is used to declare an object's property. When used, the class must have two private methods in the form of TYPE get()/void set(TYPE) for setting and getting the property. By using this macro, the object gets a property interface for managing the value is if it was a field. Arithmetic and other operators work as expected. The defined property is reflected: it can be examined and set in run-time. Properties do not define an access attribute: there are always public. The property does not add any extra bytes to the class, besides those needed for reflection.
Parameters:
TYPE - type of the property
NAME - name of the property
