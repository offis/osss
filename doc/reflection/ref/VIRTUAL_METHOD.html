The VIRTUAL_METHOD macro is used to declare a reflected method that is virtual, ie. can be overloaded by subclasses.
Parameters:
ACCESS_ATTR - access attribute (public, protected or private)
VIRTUAL - pass virtual or no_virtual
RETURN_TYPE - the return type of the method
METHOD_NAME - name of the method
METHOD_ARGS - the method's arguments enclosed in parentheses
